基于hexo+github框架搭建个人博客搭建完成阶段的个人总结

操作

  1. 下载node.js并安装
  2. sudo su 切换到root用户
  3. npm install -g cnpm –registry=https://registry.npm.taobao.org
  4. cnpm install -g hexo -cli
  5. cnpm install -g hexo -cli
  6. mkdir blog
  7. sudo hexo init
  8. hexo s
  9. hexo n “.md”
  10. vim .md
  11. shift zz
  12. github 新建仓库
  13. blog下 cnpm install –save hexo -deployer-git
  14. vim _config.yml
    *Deployment
    *deploy:
    *type: git
*tepo: github地址wzf159.github.io.com
*branch: master
  15. hexo clean
  16. hexo g
  17. hexo d
  18. git clone url

步骤

  1. 下载nodejs
  2. 安装cnpm
  3. 全局安装hexo
  4. 创建blog文件夹
  5. 管理员权限下安装
  6. 启动 localhost下观察是否启动
  7. 新建md文件编辑内容，保存退出
  8. blog安装git
  9. 设置_config.yml配置文件，设置github网址
  10. 清理 上传
